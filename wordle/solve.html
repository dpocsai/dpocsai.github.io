How to use:

1) WORDLE: Submit a starting guess and remember tile colors

2) SOLVER: Type in the same starting guess

3) SOLVER: Tap each box to set color

4) SOLVER: Tap the Guess button to generate the optimal next guess.

5) WORDLE: Submit the generated guess and remember tile colors

6) Repeat steps 3-5 until complete!
